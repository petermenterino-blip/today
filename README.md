# Mentorino - Premium Mentorship Platform

A premium career, education, and life guidance mentorship platform.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` in `.env.local`
3. Run the app:
   `npm run dev`
