## How to Export SQL from Supabase

### Option 1: Supabase CLI (recommended)
```bash
# Install Supabase CLI
# https://supabase.com/docs/guides/cli

# Link to your project
npx supabase login
npx supabase link --project-ref jnazlfhhzxrocvxvmkkc

# Full database dump
npx supabase db dump -f backups/supabase_schema_v1.sql

# Data only
npx supabase db dump --data-only -f backups/supabase_data_v1.sql

# Functions and triggers
npx supabase db dump --schema-only -f backups/database_functions_v1.sql
```

### Option 2: Supabase Dashboard SQL Editor
1. Go to https://supabase.com/dashboard/project/jnazlfhhzxrocvxvmkkc
2. Open SQL Editor
3. Run: `SELECT 'CREATE TABLE ' || tablename || ' (...);' FROM pg_tables WHERE schemaname = 'public';`
4. Copy results to file
5. For complete dump, use pg_dump-style export in Settings → Database → Database Settings

### Option 3: pg_dump (requires connection string)
```bash
pg_dump --dbname=postgresql://postgres:[password]@db.jnazlfhhzxrocvxvmkkc.supabase.co:5432/postgres --schema-only -f backups/supabase_schema_v1.sql
```
